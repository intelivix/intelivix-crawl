Installation Guide
=============================

Dependencies
------------------

    * `Python 2.7 <http://www.python.org/>`_.
    * `pip <https://pip.pypa.io/en/latest/installing/>`_.
    * `scrapy <http://doc.scrapy.org/en/latest/index.html/>`_.



Pip Install
-------------

For install with pip:

.. code-block:: bash
    
   $ pip install scrapy-venom
